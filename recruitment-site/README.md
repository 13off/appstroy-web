# СКБС Работа

Отдельный статический сайт найма для ООО «СКБС». Проект не связан с исходниками AppСтой и не изменяет приложение.

## Что готово

- адаптивный чёрно-золотой лендинг;
- карточки вакансий;
- блок условий и этапов трудоустройства;
- форма кандидата с валидацией;
- Netlify Forms без собственного backend;
- страница успешной отправки;
- черновик политики конфиденциальности;
- плавные анимации и поддержка reduced motion.

## Бесплатная публикация на Netlify

1. Подключить репозиторий в Netlify.
2. Выбрать ветку `agent/recruitment-site`.
3. Base directory: `recruitment-site`.
4. Build command оставить пустым.
5. Publish directory: `.`.
6. После первого деплоя Netlify автоматически распознает форму `job-application`.

## Перед публичным запуском заменить

- точные вакансии и ставки;
- актуальные объекты и даты заезда;
- контакты;
- ИНН, юридический адрес и реквизиты;
- финальный текст политики конфиденциальности;
- реальные фотографии объектов и проживания.
