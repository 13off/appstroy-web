# СКБС Работа

Отдельный статический сайт найма для ООО «СКБС». Проект не связан с исходниками AppСтрой и не изменяет приложение.

## Опубликовать бесплатно

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/13off/appstroy-web&branch=agent/recruitment-site&create_from_path=recruitment-site)

Кнопка создаёт отдельный репозиторий только с сайтом найма и публикует его на Netlify. Исходники AppСтрой в новый репозиторий не копируются.

После публикации:

1. Открыть в Netlify раздел **Forms**.
2. Убедиться, что появилась форма `job-application`.
3. Отправить тестовую заявку с сайта.
4. При необходимости включить уведомления о новых заявках.

## Что готово

- адаптивный чёрно-золотой лендинг;
- карточки вакансий;
- блок условий и этапов трудоустройства;
- форма кандидата с валидацией;
- Netlify Forms без собственного backend;
- страница успешной отправки;
- черновик политики конфиденциальности;
- плавные анимации и поддержка reduced motion;
- безопасные HTTP-заголовки через `netlify.toml`.
